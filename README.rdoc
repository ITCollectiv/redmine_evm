= redmine_evm

V1.0.0 (4-Jul-2014)


==About

  This plugin uses EVM (Earned Value Management) a project monitoring methodology
  that measures the current progress of the project to generate three key metrics,
  the planned value, actual cost and earned value. 
  
  With this plugin you can:
  -Manage and keep an history of baselines.
  -Get information about the current project performance.
  -Predict the project final cost and duration.
